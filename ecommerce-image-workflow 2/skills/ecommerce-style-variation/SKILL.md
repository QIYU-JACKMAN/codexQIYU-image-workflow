# 电商风格裂变

保持内联执行，不使用子代理，不创建素材投递文件夹。

## 对话输入

用户可直接上传 1-5 张图片和需求。先区分风格参考图与商品事实图；总输入最多 5 张。只有图片没有文字时，只询问希望裂变的数量和用途。

## 工作流

1. 定位插件根目录并读取公共 `gpt-image-2-prompt-optimizer.md`、项目 schema 和质检规则。
2. 提取参考图的视觉 DNA：配色、光线、镜头、材质、排版、留白、氛围和商业用途，不复制参考品牌与商品事实。
3. 有商品图时单独锁定商品结构和可见细节。
4. 根据用户数量规划 1-10 个明显不同的构图方向，每个任务只改变一个主要构图维度。
5. 创建 `style-variation` 项目，所有任务保持已确认的输入图片顺序，使用内置优化器生成独立 `style-transfer` 或 `ads-marketing` 提示词。
6. 展示视觉 DNA、各构图方向、优化提示词和图片映射，请客户校验。允许删除或修改单个方向。
7. 全部启用任务确认后生成，检查风格一致性、构图差异和商品身份；只重画失败方向。
