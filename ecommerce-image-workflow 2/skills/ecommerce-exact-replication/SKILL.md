# 电商一比一复刻

保持内联执行，不使用子代理，不创建素材投递文件夹。

## 对话输入

允许用户在一条消息中直接上传参考图、0-4 张同一商品的事实图和文字要求。已有信息不重复询问。图片职责不明确时，先展示判断结果并让用户确认。

固定映射：`Image 1` 是参考图，后续图片是同一商品的不同角度或细节。多个不同 SKU 应转到批量 SKU。

## 工作流

1. 分析商品图的结构、颜色、Logo、包装、配件和易画错细节。
2. 独立分析参考图的构图、镜头、光线、版式、文字层级和主体比例。参考品牌、商品事实和原文案默认不复用。
3. 读取 `gpt-image-2-prompt-optimizer.md`，生成 `style-transfer` 或 `compositing` 提示词，明确 Image 1 提供版式和视觉结构，后续图片提供商品身份。
4. 展示原需求、优化提示词、图片映射、必须保持和默认移除内容，请客户校验。
5. 客户确认提示词和参数后创建一个已批准任务并生成。
6. 检查构图相似度、商品准确性、参考品牌泄漏、文字和额外对象；重画只做单点修改。
